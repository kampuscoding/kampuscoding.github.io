Program Magang Super Bootcamp dan Bimbingan Karir

Definisi

Program Magang Super Bootcamp merupakan program magang khusus yang diselenggarakan oleh CODEPOLITAN untuk membantu mempersiapkan lulusan universitas agar siap menghadapi dunia kerja.

Program Magang Super Bootcamp diselenggarakan 2 kali dalam setahun oleh CODEPOLITAN, dengan jumlah peserta terbatas yang telah melalui seleksi terlebih dahulu dalam program Kampus Coding.

Peserta

Peserta merupakan pelajar atau mahasiswa yang terdaftar sebagai siswa di program Kampus Coding yang telah menyelesaikan masa belajarnya dan telah lulus dalam seleksi peserta Program Magang Super Bootcamp.

Biaya Program

GRATIS.

Kegiatan

Program terdiri dari berbagai kegiatan sebagai berikut:

  • Coding Bootcamp
  • Mentoring
  • Tugas Harian
  • Bimbingan Karir
  • Proyek Tugas Akhir

Waktu & Tempat

Kegiatan berlangsung selama 2 bulan.

Lokasi kegiatan di Kantor CODEPOLITAN, Jalan Cipedes Tengah 1 No.27 Sukagalih Sukajadi Kota Bandung, Jawa Barat.

Seleksi

Seleksi diselenggarakan dalam waktu dan tempat yang ditentukan oleh CODEPOLITAN. Informasi mengenai seleksi akan ditampilkan di halaman dashboard peserta program Kampus Coding jika pendaftaran seleksi telah dibuka.
